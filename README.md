# ChatMEAN

## Installation
- Create `.env` file

- Run in terminal
```
cd ChatMEAN/ && npm i
```
```
cd ANGlient/ && npm i
```
```
cd ../ && npm run build
```

- Open in browser
```
http://localhost:[PORT]/
```
